Notes (2) of jekyll develop -- Important Git commands
by QI Wenfei – tags:

1. TAKES ME ONE AFTERNOON TO FIGURE THIS OUT.

2. git 初始化 /恢复

2.1 删除.git 文件夹 ( or 重新编译site and blog )

		$ rm -rf .git

2.2 删除所有 git 分支 

		$ git init

2.3 创建本地gh-pages (将来这个本地分支的名字会在远程创建） 所以这里就决定了远程branch的名字

		$ git checkout -b gh-pages

2.4 添加整个文件夹进commit区（working directory)

		$ git add .

2.5 为远程git地址创建本地通道 次通道会在内存里保存很久 如果要有新的源代码 无比创建新的名字

		$ git remote add pipeline2 yourgitAddress

2.6 commit with comments

		$ git commit -m "comments"

2.7 通过pipelines push gh-pages这个分支中的东西

		$ git push pipeline2 gh-pages

3. 更新 

		$ git commit -am "comments"

4. 删除远程branch 

		$ git push pipeline  --delete gh-pages

（如果不能删 则说明此branch在setting 中被设为正在使用branch，去setting 修改成其他branch）

5. 本地的pipeline 不能起一样的名字 即使你init了以后 git 还是会记录在你的缓存里面 所以 一定要起一个另外的名字 命名为pipeline2!!!

(花了一个半个下午加半个晚上发现这个问题 因为用de一个pipeline）

4. 删除远程branch 

		$ git push pipeline  --delete gh-pages

10 几个常用命令

10.1 To remove a local branch from your machine: 

		$ git branch -d the_local_branch

10.1 To remove a remote branch: 

		$ git push origin :the_remote_branch

10.2 查看current本地分支 : 

		$ git branch -d the_local_branch

10.3 创建新的本地分支: 

		$ git checkout -b newName
